# Contributing

Thanks for contributing to this project.

## GitHub-only workflow

You can complete the full contribution flow without leaving GitHub:

1. Open/create an issue that describes the change.
2. Create a branch from the issue.
3. Implement changes:
   - Small edits: use the GitHub web editor (`.` key or file edit button).
   - Larger edits: use GitHub Codespaces.
4. Commit to your branch and open a pull request.
5. Complete the PR checklist and request review.
6. Merge after approvals.

## Pull request expectations

- Keep changes focused and small.
- Link the related issue in the PR description.
- Describe what changed and how it was validated.
